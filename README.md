FoodHut - Modern Food Delivery Website 🍔
FoodHut is a professional and responsive landing page designed for a food delivery service. It provides a modern user experience using HTML, CSS, Bootstrap, and JavaScript.

🚀 Live Demo
https://foodhut--website.web.app

🔗 GitHub Repository
https://github.com/Sadi2345-Developer/foodhut-website

✨ Key Features
Fully Responsive: Works perfectly across mobile, tablet, and desktop devices.

Dynamic Menu: Food items are rendered dynamically using JavaScript arrays.

Modern UI: Features eye-catching colors, floating images, and attractive call-to-action buttons.

Bootstrap 5: Utilizes the latest components and a robust responsive grid system.

Search Bar: Includes a clean and functional search bar interface.

🛠️ Tech Stack
HTML5 - For semantic web structure.

CSS3 - For custom styling and layout design.

Bootstrap 5 - For responsive components and rapid UI development.

JavaScript - For dynamic content rendering and interactivity.
